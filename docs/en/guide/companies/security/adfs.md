# Active Directory

::: tip Microsoft ADFS
Você pode permitir a autenticação segura através do Microsoft Active Directory Federation Services. Saiba mais em: [ADFS](../../integrations/adfs/)<br>
Se precisar de apoio adicional converse com o [PhishX Support](mailto:support@phishx.io).
:::

Gerenciar a autenticação segura com o seu Active Directory.

1. In **Settings** -> **Active Directory**.

   In **URL** definir o endereço público do seu ambiente.

   In **Client ID** definir o seu ID do Cliente.

   In **Client Secret** definir o seu Segredo do Cliente.
