# Filtro de conexão

# Resumo

  1. Editar o seu Filtro de Conexão padrão em proteção -> filtro de conexão. Adicionar nossa lista atual de IPs: IPs

# Configurações

  1. Acessar o atual Microsoft 365 Exchange admin center.

    Exchange Admin Center (opens new window)

  2. Ir para filtro de conexão in menu: proteção -> filtro de conexão.

    Selecionar o seu Padrão. Clicar Editar.

    Referência (opens new window)

  3. Em filtragem de conexão -> Lista de IP Permitidos.

    Clicar +.

    Adicionar nossa lista atual de IPs: IPs.

  4. Clicar Salvar.

  5. Clicar Ok.
