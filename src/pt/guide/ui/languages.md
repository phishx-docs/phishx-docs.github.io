# Alterar idioma

   ::: tip Idioma
   A plataforma por padrão identifica o idioma do seu navegador e já altera para o idioma disponível.
   :::

   Dentro do Menu do Usuário, clicar em **Idioma**.

   Escolher o novo idioma.
