# Cambiar idioma

::: tip Idioma
La plataforma identifica por patrón el idioma de su navegador y ya cambia al idioma disponible.
:::

Dentro del Menú del Usuário, hacer clic en **Idioma**.

Elegir el nuevo idioma.
